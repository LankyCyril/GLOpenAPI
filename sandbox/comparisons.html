Query non-NA columns

Before: (think colA≠NA & colB≠NA) ?study.factor value.altitude &study.factor value.time Same syntax after: (shows columns even if there's no data) ?study.factor value.altitude &study.factor value.time Equivalent syntax after: (think colA≠NA & colB≠NA) ?=study.factor value.altitude &=study.factor value.time

Include columns with NAs while querying for others that are non-NA

Before: Impossible Same syntax after: Not applicable Correct syntax after: (think "any colA" & colB≠NA) ?study.factor value.altitude &=study.factor value.time

Include columns with NAs while querying for specific values in others

Before: Impossible Same syntax before: (subsets to non-NA in colA) ?study.factor value.altitude &study.factor value.time=day 0 Correct syntax after: (think "any colA" & colB=X) ?study.factor value.altitude &study.factor value.time=day 0

Querying within one assay is unaffected by these changes
